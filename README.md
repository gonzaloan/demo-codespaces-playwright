# Lab 01 - Playwright eCommerce Testing

## Quick Start

### In Codespaces
1. Click **Code** → **Codespaces** → **Create codespace on main**
2. Wait for setup to complete
3. Read the exercise in `docs/instructions.md`
4. Code your solution in `src/`
5. Run `npm test` to verify locally
6. Push to trigger automatic grading

## Commands

| Command | Description |
|---|---|
| `npm test` | Run your tests |
| `npm run test:report` | Open HTML report |
| `npm run grade` | Run grader locally |

## Submission

Push to `main`. Check the **Actions** tab for your grading results.
